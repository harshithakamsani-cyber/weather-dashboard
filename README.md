
# Weather Dashboard

A responsive Weather Dashboard web application built using **HTML, CSS, and JavaScript**.  
This project displays weather information with a clean and user-friendly interface.

## Features

-  Display weather information
-  Search weather by location
-  Responsive design for mobile and desktop
-  Clean and simple user interface
-  Fast and easy to use

## Technologies Used

- HTML5
- CSS3
- JavaScript
